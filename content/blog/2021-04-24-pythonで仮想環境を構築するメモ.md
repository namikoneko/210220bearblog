## uvicornをプログラムの中から実行する

<https://www.uvicorn.org/deployment/>

Running programmatically¶


example.py:
```


import uvicorn

class App:
    ...

app = App()

if __name__ == "__main__":
    uvicorn.run("example:app", host="127.0.0.1", port=5000, log_level="info")
```
